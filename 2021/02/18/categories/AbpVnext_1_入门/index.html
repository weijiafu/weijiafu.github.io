简介

教程内容来自于官方教程，进行个人简化，可能具体代码内容会忽略请去官方内容查看。

官方教程

遇到不明白的问题第一时间去官网查找，虽然汉化版不完整，可以先看外语版本的，用谷歌浏览器翻译一下还可以看。
Abp官网

入门-Web应用程序

设置开发环境

前提：VS2019、.Net 5.0+
安装ABP CLI：
dotnet tool install -g Volo.Abp.Cli
更新ABP CLI：
dotnet tool update -g Volo.Abp.Cli

创建新的解决方案

注意：Acme.BookStore可以替换成自己想设定的名字，但如果是学习使用不建议修改，否则后面官方提供教程的源码需要多处进行修改，修改成自己解决方案的名称。
abp new Acme.BookStore -u blazor

运行解决方案

将所有项目下的appsettings.json文件中的数据库连接替换成自己的数据库链接字符串
注意：注意：注意：这个字符串一定要检查，不要看着像以为就是了，不然一直连不上，还要记得不要忘了新建数据库。

启动以DbMigrator结尾的项目，并检查该项目的数据库链接字符串，启动成功后你就会发现数据库里面多了好多张表。
启动项目：根据选择的模板不同启动项也不同，将项目启动就可以了。
种子数据：账号：admin，密码：1q2w3E*
